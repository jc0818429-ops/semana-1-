Una ruta POST sirve para enviar datos nuevos desde tu aplicación hacia el servidor para que este los guarde, los procese o cree algo totalmente nuevo. Para entenderlo de forma muy humana, mientras que una ruta GET es como asomarte a mirar el perfil de alguien en redes sociales, una ruta POST es el momento exacto en el que le das al botón de Publicar una foto, envías un mensaje por chat o te registras en una página. Además de servir para crear cosas (como un nuevo usuario o un comentario), es el método ideal para manejar información delicada como contraseñas o datos de pago, ya que la información viaja protegida e invisible dentro del cuerpo del mensaje (el body) y no expuesta a la vista de todos en la barra de direcciones del navegador.
